# 🐍 Proyecto Python - Hola Mundo

Este es un proyecto básico en Python que imprime un mensaje de "Hola Mundo".  
Su propósito es servir como ejemplo inicial para la configuración de un repositorio, flujo de trabajo con Git y buenas prácticas de desarrollo.

---

## 📌 Descripción

El proyecto contiene un único archivo en Python que muestra un mensaje en consola.  
Es ideal como punto de partida para:

- Aprender Git y GitHub
- Practicar flujos de trabajo con ramas (GitFlow / TBD)
- Configurar herramientas como linters o CI/CD
