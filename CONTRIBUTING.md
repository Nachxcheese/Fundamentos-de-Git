# 🤝 Contributing

Gracias por tu interés en contribuir a este proyecto 🚀

---


## 🔀 Flujo de trabajo

Este proyecto utiliza un flujo de ramas simplificado:

- `main` → versión estable (producción)
- `develop` → rama principal de desarrollo

### 📍 Proceso de trabajo

1. Los cambios se realizan en la rama `develop`
2. Se hacen commits con descripciones claras
3. Se suben los cambios al repositorio remoto
4. Se crea un Pull Request desde `develop` hacia `main`
5. Los cambios son integrados mediante merge

## 📌 Cómo contribuir

1. Cámbiate a la rama de desarrollo:

```bash
git checkout develop
git pull origin develop
git add .
git commit -m "Agrega: descripción clara del cambio"
git push origin develop
```
